# Генератор сайта Манчкин

Статический генератор для создания базы знаний по игре Манчкин.

## Структура проекта

```
munchkin-wiki/
├── build.py          # Скрипт генерации
├── template.html     # Шаблон сайта
├── rules.txt         # Файл с правилами
└── index.html        # Сгенерированный сайт (создаётся автоматически)
```

## Установка

Установи Jinja2:

```bash
pip install jinja2
```

## Использование

1. Отредактируй `rules.txt` - добавь свои правила
2. Запусти генерацию:
   ```bash
   python build.py
   ```
3. Открой `index.html` в браузере

## Формат rules.txt

Каждое правило начинается с `===` и содержит поля:

```
===
КАТЕГОРИЯ: monsters
ЗАГОЛОВОК: Название правила
ОПИСАНИЕ: Подробное описание ситуации
РЕШЕНИЕ: Итоговое решение или вывод
ССЫЛКИ: Название1|url1, Название2|url2
ТЕГИ: тег1, тег2, тег3
===
```

### Доступные категории

- `monsters` - Непотребства
- `treasures` - Сокровища
- `curses` - Проклятия
- `mechanics` - Механики игры
- `gameplay` - Геймплей и тактика

### Правила полей

- **КАТЕГОРИЯ** - обязательное, одно из значений выше
- **ЗАГОЛОВОК** - обязательное, краткое название
- **ОПИСАНИЕ** - опциональное, подробности
- **РЕШЕНИЕ** - опциональное, выделяется зелёным
- **ССЫЛКИ** - опциональное, формат: `Текст|URL, Текст2|URL2`
- **ТЕГИ** - опциональное, через запятую

## Пример правила

```
===
КАТЕГОРИЯ: monsters
ЗАГОЛОВОК: Можно ли смывать от нескольких монстров?
ОПИСАНИЕ: Спорная ситуация с множественным боем
РЕШЕНИЕ: Бросается один кубик на всех монстров
ССЫЛКИ: Форум|https://forum.com/topic, FAQ|https://faq.com
ТЕГИ: бой, смывка, монстры
===
```

## Редактирование шаблона

Если хочешь изменить внешний вид - редактируй `template.html`.

Переменные доступные в шаблоне:
- `rules` - список всех правил
- `grouped` - правила сгруппированные по категориям
- `categories` - список категорий
- `category_names` - словарь переводов категорий
